# DesignSwipe (Next.js + Tailwind v4)

This is a merged project: Next.js app with UI components imported from your Figma-generated React project.

## Run locally

```bash
npm install
# You may need these additional UI deps (shadcn / radix components):
npm install lucide-react sonner @radix-ui/react-dialog @radix-ui/react-dropdown-menu @radix-ui/react-popover @radix-ui/react-tabs @radix-ui/react-checkbox @radix-ui/react-avatar @radix-ui/react-navigation-menu @radix-ui/react-tooltip @radix-ui/react-menubar @radix-ui/react-label
npm run dev
```
Open http://localhost:3000

If Tailwind throws version errors, ensure tailwindcss ^4 and @tailwindcss/postcss are installed.
